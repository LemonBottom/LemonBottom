Python Garbage Collector

Posted on 2020-11-29

Python provides ability to control garbage collection:

Python use two method to collect garbage which are mark-sweep algorithm and generational collection algorithm .

  • Mark-sweep:

The objects that collector found to be unreachable but could not be freed would be add to a list, which is gc.garbage.

We can get the reference count of the object:

If reference count equals 0, the object will be collected automatically.

  • Generational collection:

If there is a problem a reference cycle, we can’t solve it by the reference count algorithm.Another algorithm could solve it.

The object just created is in the first generation of the garbage collector. If python executes a garbage collection process on a generation but the object survives, it moves up into a second, older generation. The python garbage collector has three generations in total.

    • 函数一结束就取返回值
